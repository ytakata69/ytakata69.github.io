man, info

  • manコマンド … ld等のインストール済みコマンドの情報

    • man コマンド名 … 例: man ld, man man

    • システムコールや標準ライブラリ関数の情報も得られる

      • 例: man 2 intro … システムコール全般に関する情報
      • 例: man 2 write … writeシステムコールの情報

    • manコマンドは1個のテキスト文書を less コマンド（ページ送りソフトウェア）を使って表示する。 qキーを押せば終了する。hキーを押せばヘルプ画面を表示する。 man less を参照。

  • infoコマンド … 一部のコマンドに対するより詳しい情報

    • 例: info nasm … NASMの情報（上記のWeb上のマニュアルと同内容）
    • 例: info ld … ld (GNU linker) の情報
    • 例: info binutils … objdump等のELF関連コマンドの情報
    • 例: info make … GNU makeの情報（上記のWeb上のマニュアルと同内容）
    • 表示される情報はハイパーテキスト（複数のページがリンクで繋がれた文書）になっている。 qキーを押せば終了する。hキーを押せばヘルプ画面が表示される。info info を参照。
